express

Είναι ένα web framework που χρησιμοποιείται για να διαχειριστούμε πολλά διαφορετικά http requests σε ένα συγκεκριμένο URL
Το express βοηθάει στο να απαντάμε σε requests με υποστήριξη routing έτσι ώστε να μπορούμε να γράψουμε responses σε συγκεκριμένα URLs

body-parser

Προκειμένου να διαχειριζόμαστε HTTP POST requests στο Express.js πρέπει να εγκαταστήσουμε το middleware module body-parser
Το body-parser αντλεί το body portion ενός request stream και το εκθέτει στο req.body

Sequelization

Sequelization είναι η διαδικασία σύνδεσης μίας Node εφαρμογής σε ένα Object-relational Mapper (ORM) για καλύτερο συγχρονισμό με τη βάση δεδομένων

Object-relational Mapper (ORM)

Βοηθάει στο data manipulation και querying μέσω της χρήσης objects από τη βάση δεδομένων
Χρησιμοποιόντας έναν ORM βελτιστοποιούμε τα SQL queries κάνοντας εύκολη την επαναχρησιμοποίηση τους
